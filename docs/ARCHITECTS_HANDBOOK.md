# Architects' Handbook: Bruce Lee Master AI
**Status**: Phase 3 (Multi-Agent Intelligence)

## 1. Core Architectural Paradigm: "Split-Brain"
The fundamental challenge of this project was the conflict between **Forensic Intelligence** (clinical data deconstruction) and **Persona Fidelity** (high-amplitude Bruce Lee voice). A single-prompt approach caused "Persona Drift," where engineering instructions muted the character.

### Multi-Agent Orchestration
We have decoupled the "Science" from the "Soul" using a sequential agentic flow:

1.  **Agent A (ForensicAnalyst.ts)**: 
    - **Focus**: Sports Science, Biomechanics, CNS Patterning, TCM.
    - **Data Input**: Archival retrieval (`KnowledgeService.ts`) and Persistent Memory (`MemoryService.ts`).
    - **Output**: A clinical, objective report (Hidden from user, visible in UI Microscope).

2.  **Agent B (PersonaAgent.ts)**:
    - **Focus**: 100% User Identity.
    - **Guidance**: Uses Agent A's report as the "Hidden Truth" to synthesize a high-amplitude response.
    - **Constraints**: No robotic tags, no JSON, addresses user as "My Friend."

## 2. Forensic Engineering Philosophy
We treat Bruce Lee's training as a dataset to be deconstructed. The system is grounded in **"The Art of Expressing the Human Body"**. Every strike is analyzed through:
- **Kinetic Chain Efficiency**: How energy transfers from the ground to the impact point.
- **CNS Tension**: Neuromuscular load and firing patterns.
- **Chi (Metaphysical Alignment)**: Energetic flow and center-of-gravity.

## 3. Persistent Memory & Self-Evolution
The **DreamTask** protocol (via AI-driven consolidation) ensures that short-term training logs are converted into long-term philosophical axioms. This creates a compounding intelligence that "remembers" previous breakthroughs.
